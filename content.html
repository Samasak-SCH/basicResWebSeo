หมายเหตุ : ที่มา  https://en.wikipedia.org/wiki/Guido_van_Rossum

ภาษาไพธอน เป็นภาษาการเขียนโปรแกรมคอมพิวเตอร์ระดับสูงที่พัฒนาขึ้นจาก Guido van Rossum และได้ถูกนำออกมาให้ใช้งานกันในปี 1991 โดยการนำจุดเด่นของภาษาต่าง ๆ เอาไว้เข้าด้วยกัน กล่าวคือ

  • การมีไวยกรณ์ของภาษาที่เข้าใจง่ายซึ่งทำให้ง่ายต่อการศึกษาและพัฒนา
  • การเขียนโปรแกรมสั้นกระชับจึงทำให้เขียนโปรแกรมได้เร็วขึ้น
  • ลดความซับซ้อนและความผิดพลาดจากการประกาศตัวแปร

ซึ่งโปรแกรมภาษาไพธอนจะทำการกำหนดชนิดและขนาดข้อมูลให้กับผู้ใช้งานโดยอัตโนมัติทำให้ง่ายต่อการทำงานและการเขียนโปรแกรม นอกจากนี้ภาษาไพธอนยังสนับสนุนการเขียนโปรแกรมในหลายรูปแบบ เช่น

  • การเขียนโปรแกรมแบบโครงสร้าง
  • การเขียนโปรแกรมเชิงวัตถุ (OOP)
  • การสร้างโปรแกรม GUI, CGI และการสร้างเกมส์
  • การติดต่อกับฐานข้อมูล
  • การทำงานผ่านระบบเครือข่าย และระบบอีเมล
  • การทำงานข้ามแพลตฟอร์ม

user input ในภาษาไพธอนเป็นการเขียนโปรแกรมให้ทำการรอรับคำสั่งจากผู้ใช้งานผ่าน command prompt โดยที่ค่าที่ได้รับเข้ามานั้นจะถูกรับเข้ามาด้วยคำสั้ง input() ผ่านทางคีย์บอร์ดซึ่งสามารถรับค่าได้ทั้งตัวอักษรและค่าตัวเลข

จากตัวอย่างดังกล่าวจะเห็นได้ว่าฟังก์ชั่น input( ) และ raw_input() เป็นการรับค่า ตัวอักษร(string ) ที่ได้มาจากคีย์บอร์ดมาเก็บไว้ที่ตัวแปร username ก่อนจะนำไปแสดงผลต่อไป ในบางครั้งผู้เขียนโปรแกรมอาจต้องการให้ค่าที่รับมานั้นเป็น ตัวเลขจำนวนเต็ม(integer) หรือ ตัวเลขทศนิยม(float) ก็สามารถทำได้เช่นเดี่ยวกัน โดยการแปลงค่าที่รับจากคีย์บอร์ดเป็นตัวแปรชนิดที่ต้องการเสียก่อน

Code command when using integer variable

จะเห็นได้ว่าฟังก์ชั่น input() จากตัวอย่างดังกล่าวนั้นมีการเปิดรับค่าที่เป็นตัวเลข ปีของผู้เกิดที่เป็น ค.ศ. เมื่อผู้ใช้ทำการใส่ปีเกิดที่เป็นตัวเลขเป็นที่เรียบร้อยแล้วโปรแกรมจะทำการคำนวณค่าที่ได้ออกมาเป็นอายุของผู้ใช้ อย่างไรก็ดีจะเห็นไดว่าโปรแกรมดังกล่าวนั้นยังคงมีจุดบกพร่องกล่าวคือ โปรแกรมไม่ได้มีการตรวจสอบค่าที่ได้รับว่าเป็นตัวเลขหรือไม่ หากผู้ใช้ป้อนตัวอักษรเข้ามาในส่วนของการระบุปีเกิดก็จะเกิดข้อ ผิดพลาด(error) และจะออกจากโปรแกรมโดยอัตโนมัติ

การแก้ไขปัญหาการรับค่าผิดพลาดด้วยฟังก์ชั่น while() และ try-except()

สำหรับการแก้ไขในเบื้องต้นสามารถทำได้โดยการเพิ่มฟังก์ชั่น while() และ try-except() ประกอบกันเพื่อทำการตรวจสอบว่าการป้อนข้อมูลนั้นเป็นตัวเลขหรือไม่ หากไม่ใช่โปรแกรมจะทำการแจ้งเตือนผู้ใช้งานให้ทราบและวนกลับไปที่จุดเริ่มต้นของโปรแกรมเพื่อให้ผู้ใช้งานป้อนข้อมูลให้ถูกต้อง

Code command when using try-exept()

โปรแกรมดังกล่าวเป็นโปรแกรมที่ได้มีการเพิ่มเติมฟังก์ชั่น while() และ try-except() เข้าไปเพื่อใช้สำหรับการตรวจสอบโดยมีการทำงานในส่วนต่าง ๆ ดังต่อไปนี้

  • ส่วนที่ 1 : รับข้อมูลชื่อที่เป็นตัวอักษรจากผู้ใช้
  • ส่วนที่ 2 : รับข้อมูลปีเกิดจากผู้ใช้ โดยใช้การวนลูปการทำงานด้วยฟังก์ชั่น while()และฟังก์ชัน try-except เพื่อตรวจสอบว่าผู้ใช้ป้อนปีเกิดเป็นตัวเลขหรือไม่ ถ้าไม่ใช่ตัวเลขโปรแกรมจะแจ้งเตือนผู้ใช้ให้ป้อนใหม่จนกว่าจะถูกต้อง
  • ส่วนที่ 3 : คำนวณอายุจากสมการ current_year - birth_year
  • ส่วนที่ 4 : แสดงผลข้อมูลด้วยฟังก์ชั่น print()
